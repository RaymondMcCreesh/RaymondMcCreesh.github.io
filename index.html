Sign Detector

The Hog-SVM detects road signs in images. A classifier is trained by first calculating a histogram of oriented gradients for each 8x8 cell in the image, these HOGs are then concatenated and passed to the Support Vector Machine classifier. The detections are sorted by scale, and overlapping is accounted for to determine best detections per sign in each case.

Adaboost classifier

Adaboost are used to increase the accuracy of weak classifiers such as decision trees or linear classifiers. These can be used to predict if a customer will pay their insurance premium, the value of an investment or to calculate life expectancy of someone.

This project is a machine learning meta-algorithm used to improve the accuracy of classifiers.

MNIST digit classifier

The MNIST dataset is a set of handrwritten single digit images. A CNN is created using Keras and trained on a set of labelled images. The models accuracy was validated on testing data, and the CNN structure modified until reaching peak performance. A final accuracy of 99.45% was achieved on testing data.
